# Security and privacy

AI Quota Bar reads local Codex metadata, session logs, process metadata and the Codex state database. It does not transmit this data.

Please do not attach raw `~/.codex` session files to public issues. They can contain prompts, file paths and other private project information. When reporting a bug, use redacted screenshots and remove task titles, home-directory paths and account identifiers.

Security reports should be sent privately to the repository owner instead of being opened as public issues.
