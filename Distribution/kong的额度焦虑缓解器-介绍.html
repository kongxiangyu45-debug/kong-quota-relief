它主要解决三件事

不讲复杂术语，只把每天真正会看的东西放在一起。

01

还剩多少

Codex 看额度百分比和重置时间。WorkBuddy 看剩余总积分，不再把两种规则混在一起。

02

谁用得多

看到近期任务、Token 处理量，以及 WorkBuddy 最近一轮和整个任务分别用了多少积分。

03

直接回任务

任务不是一行死文字。点一下，就能回到 Codex 或 WorkBuddy 里的对应任务继续处理。

先看你的电脑能不能用

现在可以用

  • M1、M2、M3、M4 等苹果芯片 Mac
  • macOS 14 或更新版本
  • 已经安装并登录 Codex
  • 已经安装 CodexBar
  • 需要 WorkBuddy 数据时，安装并登录 WorkBuddy

现在还不能用

  • Windows 电脑
  • Intel 芯片的旧款 Mac
  • 低于 macOS 14 的系统
  • 没有 CodexBar 时，无法读取 Codex 官方额度

安装方法

  1. 解压收到的 ZIP 文件

    里面会看到“kong的额度焦虑缓解器.app”。

  2. 把应用拖进“应用程序”文件夹

    以后它会作为菜单栏小工具运行。

  3. 第一次打开时，右键选择“打开”

    这是测试版，Mac 可能会先弹出安全提醒。

  4. 登录自己的 Codex 和 WorkBuddy

    工具读取的是这台电脑上的账号数据，不是制作者的数据。

看到空白或没有额度怎么办？

先确认 CodexBar 已安装并能正常显示额度。WorkBuddy 部分只有在安装并登录 WorkBuddy 后才会出现。

关于隐私，简单说清楚

压缩包里没有制作者的账号、任务、积分或登录信息。

每个人看自己的数据安装后读取的是使用者自己电脑上的 Codex 和 WorkBuddy 记录。
任务分析在本机完成任务标题、状态和本地日志不会因为安装这个工具自动发给制作者。
WorkBuddy 余额来自官方接口工具使用本机已经登录的 WorkBuddy 身份读取当前余额。
这是测试版Codex 或 WorkBuddy 更新内部规则后，个别显示可能需要同步调整。
