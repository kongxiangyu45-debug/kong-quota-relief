# kong的额度焦虑缓解器 Windows 使用说明

适用于 Windows 10/11 64 位系统。程序不需要安装，解压后直接运行。

## 怎么开始

1. 双击 `kong的额度焦虑缓解器.exe`。
2. 程序会出现在 Windows 右下角托盘里。没看到时，点击右下角的 `^`。
3. 单击托盘图标，即可查看 Codex 额度、WorkBuddy 积分和近期任务。
4. 点击任务名称，可以直接回到对应任务。

## 使用前需要什么

- Codex 已安装并登录。
- 需要 WorkBuddy 数据时，请先安装并登录 WorkBuddy。
- 在线额度来自平台接口；任务和用量明细来自当前电脑的本地记录。

## 任务有了，Codex 额度却没显示

1. 先确认 Codex 本身可以正常联网。
2. 新版会自动沿用 Windows 系统代理、PAC 和环境变量代理。
3. 刚切换代理时，退出本程序后重新打开，再点击“刷新”。
4. 如果仍然失败，重启代理软件后再试一次。

能看到近期任务，说明本地记录读取正常；只有额度不显示，通常是在线额度接口没有走到代理，不是套餐不支持。

## 第一次运行的安全提示

这是没有购买商业签名证书的测试版。如果 Windows SmartScreen 拦截，请确认文件来源可信，再选择“更多信息”和“仍要运行”。

## 怎么退出

单击托盘图标，在菜单最下面选择“退出”。
