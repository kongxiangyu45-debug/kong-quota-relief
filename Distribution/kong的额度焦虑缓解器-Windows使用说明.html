kong的额度焦虑缓解器

Windows 右下角托盘测试版

Windows 10/11 64 位 x64 无需安装 .NET 自动跟随系统代理 本机读取

怎么开始

  1. 解压 ZIP，双击 kong的额度焦虑缓解器.exe。
  2. 程序不会弹出大窗口，它会出现在 Windows 右下角托盘里。没看到时，点一下右下角的“^”。
  3. 单击托盘图标，查看 Codex 额度、WorkBuddy 积分和近期任务。
  4. 点击任务名称，可以直接回到 Codex 或 WorkBuddy 的对应任务。

使用前需要什么

  • Codex 已安装并登录。没有登录时，Codex 额度不会出现。
  • 需要 WorkBuddy 数据时，请先安装并登录 WorkBuddy。
  • 联网时读取两个平台的官方余额；任务标题和用量明细来自这台电脑的本地记录。

第一次运行的安全提示

这是未购买商业签名证书的测试版。

如果 Windows SmartScreen 拦截，确认文件来源可信后，选择“更多信息”再选择“仍要运行”。

任务有了，Codex 额度却没显示

  1. 先确认 Codex 本身可以正常联网，并且已经登录当前账号。
  2. 新版会自动沿用 Windows 的系统代理和 PAC。刚切换代理时，先退出本程序，再重新双击打开。
  3. 打开托盘菜单，点击“刷新”。如果仍然失败，重启代理软件后再试一次。

能看到近期任务，说明本地记录读取正常；只有额度不显示，通常是在线额度接口没有走到代理，并不是套餐不支持。

它会读取什么

程序只读取当前 Windows 用户自己的 Codex、WorkBuddy 登录状态和本地任务数据库。压缩包里没有制作者的账号、Token、积分或任务记录。生成的用量报告也只保存在本机临时目录。

怎么退出

单击托盘图标，在菜单最下面选择“退出”。直接关闭说明页不会退出托盘程序。
